## 其他标签元素
> p,pre,blockquote,em,strong,span,br,cite,p,code

### p元素
表示一个段落

### pre元素
- 表示预定义格式文本
  - 按照原文本中的格式进行显示
  - 空白符都会原样显示

### blockquote元素
  - 引用内容
  - 属性：cite

  ![](assets/html/images/other1.png)

### em元素
- 强调，需要用户着重阅读的内容
  - 默认斜体显示
  - 可以嵌套

### strong元素
- 重要，严重，紧急的内容
  - 标题，警告，注意事项，指示性内容
  - 默认粗体
  - 可以嵌套


### span元素
- 无任何特殊语义
  - 行内容器

### br元素
- 换行

### cite,p元素
- 引用元素

### code
- 代码元素
