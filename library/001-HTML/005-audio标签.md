## audio元素介绍
> 音频播放器

### 属性
- src：音频URL
- autoplay：音频在就绪后马上播放
- loop：当媒介文件完成播放后再次开始播放
- muted：音频的音频输出被静音
- controls：显示音频控制栏
- crossorigin：跨域
- source(src,type)：可以针对不同浏览器加载不同格式的音频文件
- track：为音频提供字幕
  - kind(subtitles,captions,descriptions,chapters,metadata)
  - src
  - srclang
  - label
  - default
